# Exemple 8 — C minimal (Makefile, sans CMake)

Variante du langage "cpp" détectée via un simple `Makefile` (pas de
`CMakeLists.txt`). Comme pour l'exemple 4, aucune métadonnée n'est
extraite automatiquement.

```bash
./scripts/dev.sh check --path examples/08-c-makefile-simple
./scripts/dev.sh build --path examples/08-c-makefile-simple
```
